## Learnings
- Google Calendar API מסתנכרן אוטומטית ל-Android. ב-iPhone צריך הגדרה חד-פעמית.
- whatsapp-web.js שומר Session ב-LocalAuth – סריקת QR רק פעם אחת.
- QR code חייב להיות תמונה (qrcode npm) ולא ASCII כשמציגים בדפדפן נייד.
- font-size: 16px על input מונע auto-zoom באייפון.
- הודעות קוליות ב-WhatsApp מגיעות כ-type: 'ptt' (push to talk).
- Claude מחזיר JSON אמין כשה-System Prompt מגדיר את הפורמט בצורה ברורה.
- conferenceData עם hangoutsMeet יוצר Google Meet link אוטומטית.
